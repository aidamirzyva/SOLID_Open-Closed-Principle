Open/Closed Principle (Açıq/Qapalı Prinsipi)
Repository Description:
"Bu repository, SOLID prinsiplərindən olan Açıq/Qapalı Prinsipini (OCP) əyani nümunələrlə göstərir. OCP-nin əsas fikri budur: 'Proqram təminatı elementləri (siniflər, modullar, funksiyalar və s.) genişlənmə üçün açıq, dəyişdirilmə üçün qapalı olmalıdır.' Sadə dildə - yeni funksionallıq əlavə etmək istəyəndə, mövcud kodu dəyişdirmək əvəzinə, mövcud kodu genişləndirməlisiniz. Bu nümunə, ödəniş sistemində müxtəlif ödəniş üsullarının necə əlavə edilə biləcəyini, mövcud kodu dəyişdirmədən göstərir."
